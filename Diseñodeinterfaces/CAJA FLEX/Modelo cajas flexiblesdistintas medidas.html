Propiedad Flex - distintos tamaños de cajas

La propiedad flex declara la caja como flexible y ayuda a distribuir el espacio entre las cajas.

Como valor, puede tomar tres posibles parámetros separados por un espacio:

  • flex-grow
  • flex-shrink
  • fles-basis

Si en la propiedad flex se da el valor 1 al parámetro flex-grow la caja es declarada como flexible. Otra cualidad de este parámetro es indicar la proporción para la expansióndel elemento.

El párametro flex-shrink declara la proporción para la reducción del elemento, o cuánto se va a encoger el elemento con respecto a sus hermanos.

El parámetro flex-shiring declara la proporción para la reducción del elemento, o cuánto se va a encoger el elemento con respecto a sus hermanos.

El parámetro flex-basis indica el tamaño inicial del elemento que será considerado cuando el espacio libre sea distribuido entre todos los hijos del contenedor.

Las cajas flexibles serán expandidas o reducidas para llenar el espacio extra dentro del elemento padre. La distribuición del espacio depende de las propiedades del resto de las cajas. Si todos los hijos son declarados como flexibles, el tamaño de cada uno dependerá del tamaño del elemento padre y del valor de la propiedad flex.

En este ejemplo todas las cajas tienen la misma anchur, ya que la propiedad flex es 1. El tamaño de cada caja es: 600x1/4=150

Si modificamos el valor de flex-grow a 2, la caja roja ocupa el doble de espacio que el resto de cajas.

Para calcular el tamaño de la caja1 la operación es: 600x2/5=120

Si establecemos el tamaño fijo de 300px de ancho de la caja 1, ésta deja de ser flexible para ser fija. Se coloca la caja fija y despúes se reparte el resto del espacio entre el resto de cajas.

Del mismo modo que podemos tener una caja con tamaño fijo, podemos tener varias. El principio es el mismo: solo el espacio libre será distribuido entre las cajas flexibles

Otro escenario posible es declarar un tamaño para el elemento pero a la vez mantenerlo flexible. Para ello, aprovechamos el resto de parámetros disponibles de la propiedad flex (flex-shrink y flex-basis)

En este caso se declaran los tres parámetros para la propiedead flexde cada caja. Todas las cajas tienen el vlaor 1 para el párametro flex-grow , declarando así la misma proporción de expansión.


El párametro flex-shirink determina el valor de proporcioón por medio del cual las cajas serán encogidas para entrar dentro del espacio disponible.


El párametro flex-basis establece un valor inicial para el ancho del elemento. Para calcular los valores de reducción o expansiónde los elementos felexibles, el valor del parámetro fles-basis es considerado. Cuanod este valor es 0 o no es declarado, el valor tomado en cuenta es el tamaño del contenido del elemento.
